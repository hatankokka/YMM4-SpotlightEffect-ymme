・概要
	ゆっくりMovieMaker4に、矩形スポットライトの映像エフェクトを追加します。
	画面全体を暗くし、指定した四角形の範囲だけを明るく表示できます。
	資料解説、ニュース風動画、注目範囲の強調などに使用できます。

・主な機能
	・スポットライト範囲の中心X/Yをpx単位で指定
	・スポットライト範囲の幅/高さをpx単位で指定
	・周囲を暗くする強度を調整
	・境界ぼかしをpx単位で調整
	・画像および動画アイテムへの適用に対応

・導入方法
	・方法1
		インストーラー(SpotlightEffect.ymme)を用いて導入してください。
		ただし、この方法を利用するにはYMM4のバージョンが4.29.0.0以降かつ、
		YMM4用拡張子を関連付けしている必要があります。
		導入後、YMM4を再起動してください。

	・方法2
		SpotlightEffect.dllをYMM4フォルダ\user\pluginフォルダ内に配置してください。
		環境によってはpluginフォルダが無い場合があります。
		無かった場合は自身でpluginフォルダを作成してください。

		配置例：
		YukkuriMovieMaker4\user\plugin\SpotlightEffect\SpotlightEffect.dll

・使い方
	1.YMM4を起動します。
	2.タイムライン上の画像または動画アイテムを選択します。
	3.右側の編集パネルから映像エフェクトを追加します。
	4.一覧から「スポットライト」を選択します。
	5.中心X、中心Y、幅、高さ、暗くする強度、境界ぼかしを調整してください。

・パラメータ
	中心X
		スポットライト範囲の中心X座標をpx単位で指定します。

	中心Y
		スポットライト範囲の中心Y座標をpx単位で指定します。

	幅
		スポットライト範囲の横幅をpx単位で指定します。

	高さ
		スポットライト範囲の縦幅をpx単位で指定します。

	暗くする強度
		スポットライト範囲外を暗くする強度を指定します。
		0%で暗化なし、100%で最も暗くなります。

	境界ぼかし
		スポットライト範囲の境界をぼかす量をpx単位で指定します。

・注意事項
	・本プラグインは非公式プラグインです。
	・YMM4本体および関連DLLは同梱していません。
	・複数のYMM4環境がある場合、ymmeによる導入先が意図したYMM4と異なる場合があります。
		その場合は、使用したいYMM4の「設定」→「プラグイン」→「プラグインフォルダを開く」から、
		手動でSpotlightEffect.dllを配置してください。
	・px指定のため、プロジェクト解像度が変わると見た目の位置や大きさが変わる場合があります。
